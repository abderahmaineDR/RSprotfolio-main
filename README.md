# Personal Portfolio v2

A modern personal portfolio website built with Next.js, React, TypeScript, and Tailwind CSS.  
It showcases hero, about, experience, tech stack, projects, and contact sections with smooth animations and polished UI interactions.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP
- Lenis (smooth scrolling)

## Features

- Responsive single-page portfolio layout
- Reusable section-based architecture
- Animated transitions and scroll effects
- Project showcase with visual assets
- Contact and social presence section

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Create production build
- `npm run start` - Run production server
- `npm run lint` - Run lint checks
